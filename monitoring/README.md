# Faultline Monitoring Artifacts

This directory contains operational dashboard artifacts for Faultline.

## Files

- `grafana_faultline_dashboard.json`
- `lease_risk_dashboard.png`

## Dashboard signals

- duplicate-risk spikes
- expired leases
- stale-worker rejection count
- retry queue growth
- safe-to-operate status
- claim / commit latency

## Safe claim

This repo includes Grafana-compatible dashboard artifacts and a generated lease-risk dashboard image. It does not claim production Grafana deployment.
