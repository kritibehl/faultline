# Home Automation Reliability Dashboard

Static dashboard artifact for Faultline's HomeKit-style controller/accessory reliability simulations.

Shows:

- devices online/offline
- pairing success rate
- acknowledgement latency
- reconnect count
- failover count
- duplicate commands prevented
- stale commands rejected
- network degradation recovery status

Safe claim: this is a static dashboard artifact for home-automation reliability simulation, not a production HomeKit dashboard.
